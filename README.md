## Installation Steps And Run Locally
1. Clone the repository
    ```sh
    git clone https://github.com/shivamuttpal/Peddal-connect.git
    ```

1. Navigate backend and install Dependency
    ```sh
    cd  pedal-backend
    npm i
    nodemon server.js
    ```
1. Navigate frontend and install dependency
    ```sh
    cd  pedal-connect
    npm i
    npm run dev
    ```
## Pre-requisites
1. Node JS >= 18
1. NPM >= 9
1. Mongoose >= 8.2.3
1. Cors >= 2.8.5
